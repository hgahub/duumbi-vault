What changed & why.

Palette & contrast
Canvas is --color-ink (#1a1a18), text is --color-parchment (#f5f2ea). Brand rust and blue are kept in the same hue family but lifted for dark-surface contrast: rust #d07a47 (5.4:1 AA), blue #7ba8e0 (7.1:1 AAA). Original site values (#8c3d1d, #1d4a8c) fail WCAG on this canvas at 2.3:1 and 2.0:1.
Typography
Same as the site: DM Serif Display for display italics, Inter for UI text, DM Mono for the terminal itself.
REPL-01 · Header Hint
Brand and version given primary weight; the exit hint reads as affordance via a keycap, not prose. /help picks up the rust accent.
REPL-02 · Empty-State Tip
Card with a rust left-border, echoing the site's architecture step cards. Two-column item list: /intent create is the primary path; direct natural language is the fallback.
REPL-06 · Mode Strip
Active mode becomes a blue pill with a pulsing dot. Right-side intent slot: shows the focused intent slug in --blue-mid when present, an em dash when empty.
REPL-07 / 09 · Separators
Dot-terminated hairlines instead of full-bleed lines — structural but quiet.
REPL-08 · Prompt Input
Bordered well with a rust focus ring — matches the brand's primary call-to-attention colour.
REPL-10 · Status Bar
Linear string broken into labelled slots (time · cwd · workspace · activity). Workspace uses rust. The activity slot is empty when the app is idle; when work starts, a rust INTERRUPT pill appears with a live Braille spinner. Clicking it cancels the current work — same signal as Ctrl+C. Use the toggle in the heading to preview.
Activity vs. REPL-04 overlay
Deliberately a different signal from the existing REPL-04 Activity Spinner Overlay. REPL-04 answers "what is happening now" (build, LLM, command name) over the transcript. REPL-10's INTERRUPT button answers "give me a way to stop it" — always in the same place, never hidden behind a keystroke.
