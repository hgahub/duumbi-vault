## Intent

> Build a calculator with add, subtract, multiply, divide functions that work on i64 numbers

---

## Acceptance Criteria

- [ ] `add(a, b)` returns the sum of two i64 numbers
- [ ] `subtract(a, b)` returns the difference of two i64 numbers (a - b)
- [ ] `multiply(a, b)` returns the product of two i64 numbers
- [ ] `divide(a, b)` returns the integer quotient of two i64 numbers (a / b)
- [ ] All functions accept exactly two i64 parameters and return an i64
- [ ] All functions are defined in a dedicated `calculator/ops` module
- [ ] The `app/main` module is updated to import and call each calculator function
