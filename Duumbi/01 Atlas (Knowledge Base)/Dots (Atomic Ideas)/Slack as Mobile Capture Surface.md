# Slack as Mobile Capture Surface

## Summary

Slack is DUUMBI's quick mobile surface for ideas, issues, approvals, status checks, and agent follow-up. It is not the durable source of truth.

## Why it matters

Slack is optimized for speed and conversation, not durable retrieval. Keeping decisions only in Slack creates knowledge loss and execution drift.

## DUUMBI usage

- Capture quick ideas or bugs in Slack.
- Convert execution items into GitHub issues or Project items.
- Convert durable knowledge into Inbox or Atlas notes.
- Use Slack to monitor Oz runs and approve follow-up, then link final outputs into GitHub or Obsidian.
