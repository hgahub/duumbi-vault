Vezető szoftver fejlesztő vagy, Senior Rust fejlesztői tapasztalattal. Nézd át az Obsidian alatt található tag:project/duumbi jelölt dokumentációt. Tervezd meg a "Milestone 7 — Registry & Distribution" feladat kivitelezését. 
Készíts részletes feladat listát amit helyezz el az implementáció megkezdése előtt a GitHub alatt található "Duumbi project" projekt alatt. A feladatokat GitHub CLI segítségével vezesd az implementáció alatt. 
Ez elvárt feladata lista legyen tiszta, érthető és tesztelhető. Ha kell inkább kérdezz és gondold át nagyon alaposan. Nézd meg eddig mi készült el a projektből és onnan építkezz. Ha kell kérdezz. Miután kész a terv készíts fejlesztői ágat.
Implementációs folyamat: 
- Válaszd ki a soron következő feladatot. 
- Tisztázd a szándékot és tervezd meg 
- Készíts teszt tervet
- Implementáld
- Ellenőrizd
- Teszteld le
- Futass egy code review agent-et, javítsd az észrevételeket
- Értékeld a feladatot egy felhasználó agent segítségével, aki egy felhasználó szemével használatba veszi a funkciót és szeretne egy hasznos, produktív megoldást. 
- Tarstd karban a CLAUDE.md leírót és a memóriát
- A tanultak alapján menny mégegyszer végig a folyamaton.
